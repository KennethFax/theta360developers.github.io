API Guide

Use the RICOH THETA S API, a HTTP API based on Open Spherical Camera (OSC) specifications. Beginners should start here. Web browser using advanced HTTP clients, command line curl, Python. Planned examples for JavaScript and other languages.

Firmware upgrade, tripod hacks to access USB power port, WiFi signal tests, TCP/IP network packet loss tests. Hacks to control THETA with inexpensive boards such as Raspberry Pi using both GPIO pins and TCP/IP triggers from a secondary network interface.
